# Technologies

## Programming Languages
- **JavaScript / Node.js**: RESTful APIs, backend, async manipulation, crawlers, and automations.
- **TypeScript**: Backend development with static typing for greater safety and maintainability.
- **Python**: Automation, ETL, scripts with Selenium and pandas, web scraping, and Flask backends.
- **C# (.NET Core)**: APIs, Entity Framework for ORM, and efficient database access.
- **SQL**: Complex queries, relational modeling, triggers, seeds, and performance optimization.
- **HTML / CSS**: Interfaces and layouts for web projects and documentation.

## Frameworks and Libraries
- **AdonisJS**: Full-stack MVC framework for Node.js, with native ORM, authentication, and layered structure support.
- **Express.js**: Minimalist framework for RESTful APIs in Node.js.
- **React / Vite / Tailwind CSS**: SPA frontend with optimized build and utility-first styling.
- **React Native**: Cross-platform mobile development with multi-tenant flow support.
- **Flask**: Lightweight Python backend for APIs and analytical dashboards.
- **Entity Framework Core**: ORM for .NET Core with object-relational mapping.
- **Prisma ORM**: Modern typed ORM for Node.js/TypeScript, with migrations and query builder.
- **Sequelize**: ORM for Node.js with connection pooling, timeouts, and multiple relational databases support.
- **Selenium / Puppeteer**: Web navigation automation for scraping and testing on static and dynamic pages.
- **Cheerio**: Fast HTML parser for data extraction in Node.js.
- **Nodemailer**: Automated email sending in Node.js applications.
- **pandas**: Tabular data manipulation and analysis in Python.

## Databases
- **PostgreSQL**: Advanced relational database with triggers, indexes, and complex feature support.
- **MySQL**: Traditional relational database, query optimization and indexes.
- **SQLite**: Lightweight database for local storage in smaller projects or testing.
- **NEON**: Serverless PostgreSQL for scalable backends without infrastructure management.

## Infrastructure and Tools
- **Docker**: Containerization for isolated environments and consistent deployments.
- **GitHub Actions / CI/CD**: Automated integration and continuous delivery pipelines.
- **Azure DevOps**: Agile planning with boards, pipelines, and sprint control.
- **Git / Gitflow**: Version control with strategic branching for team organization and collaboration.
- **Swagger / OpenAPI**: Automatic and interactive documentation for RESTful APIs.
- **JWT / Cookie**: Token-based authentication and authorization for REST APIs.
- **RBAC**: Role-based access control in multi-user systems.
- **Power BI (DAX, M)**: Analytical dashboard creation, ETL, and data visualization.

## Practices and Methodologies
- **Clean Architecture**: Layer separation (Controller → Usecase → Repository) for modular, testable, and scalable code.
- **SOLID**: Object-oriented design principles applied in API and system development.
- **MVC**: Architectural pattern applied in projects with AdonisJS and similar frameworks.
- **Code Review**: Collaborative code review via PRs for quality assurance and best practices.
- **Seeds and Migrations**: Automation of initial database population and schema versioning.
- **ETL Automation**: Data pipelines for efficient collection, transformation, and loading.
