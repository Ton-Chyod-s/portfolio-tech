# Tecnologias

## Linguagens de Programação
- **JavaScript / Node.js**: APIs RESTful, backend, manipulação assíncrona, crawlers e automações.
- **TypeScript**: Desenvolvimento backend com tipagem estática para maior segurança e manutenibilidade.
- **Python**: Automação, ETL, scripts com Selenium e pandas, web scraping e backends com Flask.
- **C# (.NET Core)**: APIs, Entity Framework para ORM e acesso eficiente ao banco de dados.
- **SQL**: Consultas complexas, modelagem relacional, triggers, seeds e otimização de performance.
- **HTML / CSS**: Interfaces e layouts para projetos web e documentação.

## Frameworks e Bibliotecas
- **AdonisJS**: Framework MVC full-stack para Node.js, com suporte nativo a ORM, autenticação e estrutura em camadas.
- **Express.js**: Framework minimalista para APIs RESTful em Node.js.
- **React / Vite / Tailwind CSS**: Frontend SPA com build otimizado e estilização utilitária.
- **React Native**: Desenvolvimento mobile multiplataforma com suporte a fluxos multiempresa.
- **Flask**: Backend leve em Python para APIs e dashboards analíticos.
- **Entity Framework Core**: ORM para .NET Core com mapeamento objeto-relacional.
- **Prisma ORM**: ORM moderno e tipado para Node.js/TypeScript, com migrations e query builder.
- **Sequelize**: ORM para Node.js com suporte a pool de conexões, timeouts e múltiplos bancos relacionais.
- **Selenium / Puppeteer**: Automação de navegação web para scraping e testes em páginas estáticas e dinâmicas.
- **Cheerio**: Parser HTML rápido para extração de dados em Node.js.
- **Nodemailer**: Envio automatizado de e-mails em aplicações Node.js.
- **pandas**: Manipulação e análise de dados tabulares em Python.

## Banco de Dados
- **PostgreSQL**: Banco relacional avançado com triggers, índices e suporte a features complexas.
- **MySQL**: Banco relacional tradicional, otimização de queries e índices.
- **SQLite**: Banco leve para armazenamento local em projetos menores ou testes.
- **NEON**: PostgreSQL serverless para backends escaláveis sem gerenciamento de infraestrutura.

## Infraestrutura e Ferramentas
- **Docker**: Containerização para ambientes isolados e deploy consistente.
- **GitHub Actions / CI/CD**: Pipelines de integração e entrega contínua automatizados.
- **Azure DevOps**: Planejamento ágil com boards, pipelines e controle de sprints.
- **Git / Gitflow**: Versionamento com branching estratégico para organização e colaboração em equipe.
- **Swagger / OpenAPI**: Documentação automática e interativa de APIs RESTful.
- **JWT / Cookie**: Autenticação e autorização baseada em tokens para APIs REST.
- **RBAC**: Controle de acesso por papéis em sistemas multiusuário.
- **Power BI (DAX, M)**: Criação de dashboards analíticos, ETL e visualização de dados.

## Práticas e Metodologias
- **Clean Architecture**: Separação de camadas (Controller → Usecase → Repository) para código modular, testável e escalável.
- **SOLID**: Princípios de design orientado a objetos aplicados no desenvolvimento de APIs e sistemas.
- **MVC**: Padrão arquitetural aplicado em projetos com AdonisJS e frameworks similares.
- **Code Review**: Revisão colaborativa de código via PRs para garantia de qualidade e boas práticas.
- **Seeds e Migrations**: Automação de população inicial de banco e versionamento de schema.
- **Automação ETL**: Pipelines de dados para coleta, transformação e carga eficientes.
